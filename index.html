Get a live URL for each project in 5 minutes
Run these commands for each project folder. Replace YOURUSERNAME with your GitHub username. Each repo gets its own live URL you can put on your resume.

2Open terminal, navigate to the folder containing these project folders, then run:
# Run once per project — change the folder and repo name each time cd 01-azure-cost-copilot git init git add . git commit -m "Azure Cost Copilot — PM portfolio" git branch -M main git remote add origin https://github.com/YOURUSERNAME/azure-cost-copilot.git git push -u origin main cd ..
3Enable GitHub Pages: go to the repo → Settings → Pages → Source: Deploy from branch → main → / (root) → Save. Wait ~60 seconds.
4Your live URL: https://YOURUSERNAME.github.io/azure-cost-copilot — paste this on your resume and LinkedIn.
5Repeat for all 6 projects + this landing page. Create a 7th repo called pm-portfolio and push the 00-landing folder into it.
